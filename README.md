## 📦 Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/stock-dashboard.git
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
- Create a `.env` file
- Add necessary configurations

4. Run the application
```bash
npm run dev
```
